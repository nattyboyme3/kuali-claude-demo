# Kuali Build Tools

A collection of Python scripts for managing Kuali Build apps and documents.

---

## Tools

### [Kuali Document Cleaner](README_document_cleaner.md)

Finds and permanently deletes documents in Kuali Build that were submitted before a date you choose. Useful for cleaning up test data or archiving old submissions.

### [Kuali Creator Tracker](README_creator_tracker.md)

Monitors who is creating apps in Kuali Build. Compares the current list of app creators against the last run and reports any new creators, with optional email notifications.
